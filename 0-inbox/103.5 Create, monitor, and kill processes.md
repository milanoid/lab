`xeyes`, `xclock`

`CTRL-c` vs `CTRL-z`

`CTRL-c` - breaks the program
`CTRL-z` - stops the program (xeyes window still up)

```
milan@ubuntu:~$ xeyes 
^C
milan@ubuntu:~$ xeyes 
^Z
[1]+  Stopped                 xeyes

```
