## Adding context

Three common CLAUDE.md files locations:

### CLAUDE.md

- project level
- generated by `/init`
- commit to source code
- shared with others

### CLAUDE.local.md

- not shared with others
- personal instructions and customizations for Claude

### ~/.claude/CLAUDE.md

- used with all projects on my machine
- contains instructions y ou want Claude to follow on all projects


### memory mode

```
# Use comments sparingly. Only comment complex code.
```

- the `#` pound sign switch to memory mode
- updates CLAUDE.md file

### file mentions

- `@` - refer to a file 
- add to context
